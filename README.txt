## Description

This plugin allows you to add Sendlio tracking code into your website.

In addition to basic worpress compatability, the plugin also supports WooCommerce.

**What is Sendlio?**
Sendlio is a 3rd-Party service that provides email, sms and voice automation services.

This plugin adds the Sendlio javascript installation code to your website to support syncing members with Sendlio and adding embedded forms.

If WooCommerce is installed, e-commmerce related stats will also be added to your members in Sendlio (customer address, total orders, currency, total spent and dates of first and last orders). 

**Learn more:**
[Sendlio Website](https://www.sendlio.com)
[Sendlio Terms & Privacy](https://www.sendlio.com/terms)

## Installation

1. Install Sendlio by uploading the `sendlio-wordpress` directory to the `/wp-content/plugins/` directory.
2. Activate Sendlio Integration through the `Plugins` menu in WordPress.
3. Insert Sendlio company code by going to the `Settings > Sendlio` menu.
